# azk-projects-boilerplate

`azk-projects-boilerplate` follow `azk` standards to create new npm packages.

#### Compile and watch

```
$ npm start
```

#### Test + lint

```
$ npm test
```

#### Lint (jshint + jscs)

```
$ gulp lint
```

#### Test + lint and watch

```
$ gulp test
```

#### Publish a patch to npm

```
$ npm run-script patch
```
